# Jira Governance

Enterprise standards for configuring and operating Jira
to support effective Agile delivery.

## Purpose
To ensure Jira is used as a delivery enablement tool,
not an administrative burden.

## Coverage
- Board configuration standards
- Workflow governance
- Backlog hygiene practices
- Reporting and visibility guidance

## Intended Audience
- Scrum Masters
- Jira Administrators
- Engineering and Delivery leaders
